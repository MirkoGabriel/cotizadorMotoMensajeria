# cotizadorMotoMensajeria
## Requeriments
* Install node on windows.
* Install express, cors, dotenv.
```bash
npm i express cors dotenv
```

* Install nodemon.
```bash
npm i nodemon -D
```

* Install google-distance.
```bash
npm install google-distance
```

## Compile
```bash
npm run dev
```
